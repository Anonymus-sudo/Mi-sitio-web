Cuales legunajes manejo?

Html

El lenguaje de marcado de hipertexto , o HTML, es el lenguaje de marcado estándar para documentos diseñados para mostrarse en un navegador web

Css

Hojas de estilo en cascada (CSS) es un lenguaje de hoja de estilo que se utiliza para describir la presentación de un documento escrito en un lenguaje de marcado como HTML.

Javascript

JavaScript, a menudo abreviado como JS, es un lenguaje de programación que cumple con la especificación ECMAScript. JavaScript es de alto nivel, a menudo compilado justo a tiempo y con múltiples paradigmas.

Php

PHP es un lenguaje de secuencias de comandos de propósito general especialmente adecuado para el desarrollo web.
